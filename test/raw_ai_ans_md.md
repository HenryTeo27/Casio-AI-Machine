# 第 16 题

设抽取 (n) 件产品，其中次品数为 (X)。

因为次品率为 (10%)，所以

[
X\sim B(n,0.1)
]

题意要求：

[
P(X>10)\ge 0.9
]

即：

[
1-P(X\le 10)\ge 0.9
]

[
P(X\le 10)\le 0.1
]

其中：

[
P(X\le 10)=\sum_{k=0}^{10}C_n^k(0.1)^k(0.9)^{n-k}
]

所以要找最小的 (n)，使：

[
\sum_{k=0}^{10}C_n^k(0.1)^k(0.9)^{n-k}\le 0.1
]

经计算：

[
n=151时,\quad P(X>10)<0.9
]

[
n=152时,\quad P(X>10)\approx 0.9031>0.9
]

所以至少应抽取：

[
\boxed{152\text{ 件}}
]

# 第 22 题

设选择甲站的人数为 (X)，则选择乙站的人数为 (100-X)。

每个人等可能选择一个车站，所以：

[
X\sim B(100,0.5)
]

设每辆车有 (m) 个座位。

要保证所有乘客有座位，需要：

[
X\le m
]

且

[
100-X\le m
]

即：

[
100-m\le X\le m
]

题意要求：

[
P(100-m\le X\le m)\ge 0.95
]

即：

[
\sum_{k=100-m}^{m}C_{100}^k(0.5)^{100}\ge 0.95
]

逐个试算：

[
m=59时,\quad P(41\le X\le 59)\approx 0.9431<0.95
]

[
m=60时,\quad P(40\le X\le 60)\approx 0.9648>0.95
]

所以每车至少要设：

[
\boxed{60\text{ 个座位}}
]
